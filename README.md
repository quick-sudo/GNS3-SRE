# GNS3 Lab Environment for SRE Practice

Лабораторный стенд для отработки навыков Site Reliability Engineering (SRE) и сетевой инженерии в среде эмулирования сетей GNS3.

- [Требования](#требования)

- [Установка и настройка](#установка-и-настройка)


## Обзор

Данный лабораторный стенд эмулирует корпоративную инфраструктуру с использованием GNS3 для отработки SRE-навыков:

- Мониторинг и алертинг
- Надежность систем и отказоустойчивость
- Автоматизация рутинных операций
- Анализ инцидентов и постмортемы
- Балансировка нагрузки и работа с трафиком

## Требования

### Аппаратные требования
- **RAM**: 8+ GB
- **ROM**: 30+ GB
- **CPU**: 4+ ядер

### Программное обеспечение
- [GNS3](https://www.gns3.com/) - платформа для эмулирования сетей
- [Ubuntu](https://ubuntu.com/) - управляющий узел
- [VirtualBox](https://www.virtualbox.org/) - для запуска виртуальных машин, в том числе GNS3 VM

## Архитектура

### Общая схема сети

Внутри сети будут работать 2 виртуальные машины пользователя: Ubuntu как "Сontrol Machine" и Ubuntu server как "Managed Machine". Также внутри сети будут маршрутизаторы, коммутаторы и VPCS.

### Компоненты инфраструктуры

| Компонент | Назначение |
|-----------|------------|
| **Cisco IOS** | Маршрутизаторы |
| **Docker** | Контейнеризированные сервисы |
| **Prometheus** | Система мониторинга |
| **Grafana** | Визуализация метрик |
| **Ansible** | Автоматизация задач |
